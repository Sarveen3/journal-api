# Journal REST API - Spring Boot

This project is a REST API built using Spring Boot.

## Features
- Create journal entry
- Get all journal entries
- Update journal entry
- Delete journal entry

## Technologies Used
- Java
- Spring Boot
- Gradle
- REST API
- Postman

## Endpoints

GET /journal  
POST /journal  
PUT /journal/{id}  
DELETE /journal/{id}
